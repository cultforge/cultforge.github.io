# CultForge 1.3.2 | Generation 2 Series

*Massive UI overhaul, pure OLED blacks, and a brand new community profile page. Download the update today to experience the fastest launcher engine we have ever built.*

### 🚀 Major Highlights

* **Massive UI Overhaul:** A completely modernized, ultra-smooth UI/UX designed to deliver top-tier functionality and a stunning visual experience. 
* **Powered by Hyperion:** We have officially replaced the legacy Kythera framework with Hyperion—our next-generation CultForge framework.
* **Aetherion 2.0 (Forge 1.20.1 Base):** We completely rewrote the display render logic. Expect double the FPS, stable performance under heavy shaders, and ultra-smooth frame rates that push all the way to your machine's max refresh rate. 

### ✨ New Features & Functionality

* **Google OAuth Integration (Mandatory):** Secure and seamless login. If your Google account is linked to a CultForge account, you will be logged in automatically. If not, the launcher will prompt you to create one natively.
* **Integrated Mod Store:** Browse and install mods directly from the launcher. (Powered by the Modrinth API—massive credits to their team!).
* **Expanded Social Hub:** Connect with the community through Global Chat, track Player Stats, and customize your brand new Community Profile Page.
* **Advanced Server Capabilities:** Added support for multiple servers, detailed server statistics, and server-wise competitive ranking.
* **Download Manager:** Take complete control over your ongoing updates and installations.
* **Push Notifications:** Stay in the loop with real-time updates and alerts directly from the launcher.

### ⚡ Performance & Under the Hood

* **Lightning-Fast Boot Time:** The launcher engine has been heavily optimized and now boots in under **5 seconds**.
* **Customizable RAM Allocation:** You can now set specific RAM allocations on a per-build basis.
* **New CDN & API Paths:** Completely overhauled backend infrastructure to provide ultra-fast download speeds.
* **Expanded Build Library:** More builds are now available to download directly from the launcher.
